# 로켓 우주여정 방치형 (단일자원·선형단계)

WAIT로 연료를 모으고 ADVANCE로 단계를 올린다. 발사대(시작) → 대기권 → 궤도 → 달 → 화성.
각 단계 도달은 `events`에 `BEAT-N`을 발동한다(서사 A겹 — 결정적 발동 로직). 화성 도달(stage 4)이면 승리.

- 자원 1개(fuel) + 단계 1개(stage)뿐, 정수 산술·랜덤 없음 = 결정적·채점가능.
- 단계 = 마일스톤 = 비트가 1:1. 이 비트 위에 나중에 B겹(삼국지/우주 대사 텍스트)을 별도 파일로 얹는다.
- 텍스트는 출력 전용, 절대 상태로 되돌아가지 않는다(엔진 결정성 보존).
