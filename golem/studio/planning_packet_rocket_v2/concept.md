# 로켓 우주여정 방치형 v2 — UPGRADE 연료효율 (누적 빌드 카드)

베이스 로켓 카드(WAIT 연료 / ADVANCE 단계)에 **UPGRADE 액션**을 얹는 누적 빌드 카드다.
기존 코드를 scratch로 다시 쓰지 않고 **수정**해서 새 규칙 한 줄(RULE-06)을 추가한다.

- UPGRADE: 연료 `upgradeCost`(5)를 써서 WAIT의 연료 효율을 1→2로 영구 상향. 초기 투자 vs 장기 효율의 선택.
- 출력 계약은 그대로다(새 키 없음). 효율(rate)은 내부 상태라 출력에 안 나온다 → UPGRADE를 안 쓰는
  기존 시나리오는 결과가 글자 하나 안 바뀐다(누적 회귀 무결).
- 텍스트/서사는 이 카드 범위 밖. 규율: 내부 효율 상태는 출력으로만 흐르지 않고 상태로만 산다.
