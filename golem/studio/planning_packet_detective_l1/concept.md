# 탐정 IF 쇼케이스 — 카드1: 집사의 거짓 알리바이

느와르 단편 인터랙티브 픽션. 베이스(detective_base)는 비 오는 밤 사무소에 들어온 의뢰로 시작해,
현장에서 단서(상처·편지·발자국)를 모아 충분하면 DEDUCTION 비트가 터지고, 지목 선택으로 결말이 갈린다.

이 카드는 현장에 **집사 심문 경로**를 더한다. 집사를 추궁하면 단서 ALIBI를 얻고, 찢긴 편지(LETTER)와
함께 손에 들어오면 알리바이가 거짓임이 드러나는 **FALSE_ALIBI 비트**가 발동한다. 기존 경로(집사를 거치지
않는 플레이)는 한 글자도 바뀌지 않는다 — 새 비트는 새 경로를 밟을 때만 발동한다.
